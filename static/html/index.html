routes/end-points


all my users routes have a base route of /api/users


all my products routes have a base route of /api/shop


This is a best practice that I have picked up


HTTP method Path Description
Get / (home) gets the home page that displays the routes

users

HTTP method Path Description
Get ^/$|^/all$ gets all the users
Get /:userID gets a user by id
Post /register creates a new user
Patch /update/:userID updates a user's data
Delete /delete/:userID deletes a user by id

products

HTTP method Path Description
Get /products gets all the products
Get /products/recent gets a list of recent products
Get /products/:prodID gets a product by id
Post /products/add creates a new product
Patch /products/update/:prodID updates product data
Delete /products/delete/:prodID deletes a product by id
